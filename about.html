About SFA

Hey there! Welcome to SFA, developed in 2021 by Radhika Choudhary & Jubin Yadav.

We created Smart Fitness Application inspired by various Machine Learning models for visual recognition .We have used POSENET a pretrained ML model that detects 17 keys points in a human Body ,these 17 keys points are used to train another model to detect different yoga poses. This application helps user perform correct yoga posture. Pose.Net, developed by Google’s TensorFlow, and ml5.js, a JavaScript library for Machine Learning on the browser. Most of my learning was following The Coding Train’s YouTube playlist on Pose Estimation.
